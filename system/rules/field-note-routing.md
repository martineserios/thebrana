---
name: field-note-routing
description: Field notes must target docs/architecture/<topic>.md. docs/reference/ is auto-generated and silently overwritten.
alwaysApply: false
---

Field notes and manual edits must target `docs/architecture/<topic>.md`.
Never write to `docs/reference/<topic>.md` — all reference files carry a
`Generated by brana reference generate — do not edit manually` banner and are
silently overwritten on the next `brana reference generate` run.

When a topic has both an architecture file (`docs/architecture/hooks.md`) and a
reference file (`docs/reference/hooks.md`), the architecture file is the only
valid target for field notes.
