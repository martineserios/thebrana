# Brana CLI

**Always prefer CLI over raw file access.** Never Read/Write/Edit `.claude/tasks.json` directly.

- `brana backlog` тАФ all task ops (get, set, query, search, add, status, roadmap, tree, tags, blocked, stale)
- `brana ops` тАФ scheduler (status, health, logs, run, enable/disable)
- `brana doctor` тАФ system health. `brana portfolio` тАФ cross-client paths.
- `brana transcribe` тАФ audio. `brana files` тАФ large files. `brana feed`/`brana inbox` тАФ content.
