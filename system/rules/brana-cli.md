# Brana CLI

**Always prefer CLI over raw file access.** Never Read/Write/Edit `.claude/tasks.json` directly.

- `brana backlog` ‚Äî all task ops (get, set, query, search, add, status, roadmap, tree, tags, blocked, stale)
- `brana ops` ‚Äî scheduler (status, health, logs, run, enable/disable)
- `brana doctor` ‚Äî system health. `brana portfolio` ‚Äî cross-client paths.
- `brana transcribe` ‚Äî audio. `brana files` ‚Äî large files. `brana feed`/`brana inbox` ‚Äî content.
