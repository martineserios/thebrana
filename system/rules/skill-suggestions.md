# Skill Suggestions

Proactively recommend skills when the moment fits. Nudge, don't nag. One suggestion per trigger.

| Trigger | Suggest |
|---------|---------|
| Starting work on a topic | `/pattern-recall [topic]` |
| Notable learning, workaround, or failure | `/retrospective` |
| End of implementation session | `/debrief` |
| After `/debrief` finds errata | `/maintain-specs` |
| After changing a rule, hook, skill, or config | `/back-propagate` |
| Stuck on a problem | `/cross-pollinate [topic]` |
| Big decision or plan | `/challenge` |
| Ready for next roadmap phase | `/build-phase` |
| First session in a new project | `/project-onboard` |
| Project being retired | `/project-retire` |
| Dimension docs might be stale | `/refresh-knowledge` then `/maintain-specs` |
| Monthly knowledge health check | `/knowledge-review` |
| Uncommitted spec changes | `/repo-cleanup` |
| Need an ADR | `/decide [title]` |
| Project needs brana alignment | `/project-align` |
| First session on a business project | `/venture-onboard` |
| Setting up business structure | `/venture-align` |
| Executing a business milestone | `/venture-phase [type]` |
| Documenting a process | `/sop [process name]` |
| Periodic business health check | `/growth-check` |

Suggest naturally in context. Example: "This session had some notable discoveries. Want to run `/retrospective` to capture them?"
