# Visual Style Guide: Blueprint / Engineering Drawing

## Identity

This looks like a technical blueprint or engineering schematic. It connects to Martin's civil engineering background — systems designed with the same rigor as a building or a bridge. Professional, precise, but not corporate.

## Visual Characteristics

- **Line quality:** Clean, precise lines. Consistent weight. Feels engineered.
- **Shapes:** Rectangles for components, circles for decision points, diamonds for gates. Standard flowchart vocabulary but with blueprint aesthetics.
- **Arrows:** Straight, orthogonal (right angles). Solid for active connections, dashed for optional or conditional flows.
- **Text:** Monospace or technical sans-serif font. Labels are crisp and short. Component names in uppercase.
- **Colors:** White/light cyan lines on deep blue background (classic blueprint). OR inverted: dark blue lines on white with a subtle grid. One accent color (amber/gold) for key components.
- **Background:** Grid pattern — like graph paper or engineering drawing paper. Subtle but visible.
- **Layout:** Structured grid alignment. Components snap to grid. Symmetry where possible.

## How to Show Before/After

- **Before:** Components placed on the grid but disconnected. Lines that start but don't reach their destination. A clear gap in the flow with a label: "SILENT BOTTLENECK — 40% patient loss."
- **After:** Same components now connected with solid lines. New components (memory layer, correction loop) added in accent color. Flow is continuous from left to right or top to bottom.
- Two separate diagrams stacked or side by side. Blueprint title block in corner: "BEFORE: inherited system" / "AFTER: designed system."

## Component Visual Language

- **The human gate:** A diamond shape (decision point) with "HUMAN GATE" label. An icon of a person or a checkmark inside.
- **The silent bottleneck:** A broken line or a component outlined in dashed red. Labeled explicitly.
- **The memory layer:** A cylinder shape (database convention) labeled "MEMORY LAYER."
- **The correction loop:** A feedback arrow that returns from output to input, labeled "CORRECTION LOOP."
- **The bridge:** A component spanning two zones (analog and digital), drawn as a connector between them.

## Mood

Engineered. "This person designs systems the way you design structures — with tolerances, specifications, and failure modes." Appeals to technical audience while signaling professionalism to founders.
