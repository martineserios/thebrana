# Visual Style Guide: Sketch / Napkin

## Identity

This looks like it was drawn by hand on paper or a whiteboard during a client meeting. Raw, personal, approachable. The kind of diagram you'd draw on a napkin to explain a system to a founder over coffee.

## Visual Characteristics

- **Line quality:** Imperfect, slightly wobbly lines. Not ruler-straight. Feels human.
- **Shapes:** Rough circles, rectangles with rounded corners drawn by hand. No perfect geometry.
- **Arrows:** Hand-drawn arrows connecting components. Some thicker, some thinner — emphasis through weight.
- **Text:** Handwriting-style font. Labels inside or next to components. Short, direct.
- **Colors:** Limited palette — black/dark gray lines on white/cream background. One accent color (warm orange or blue) to highlight key components or the transformation.
- **Background:** Slightly textured — like paper or a whiteboard. Not pure white.
- **Layout:** Organic placement. Not perfectly aligned. Components placed where they make sense spatially.

## How to Show Before/After

- **Before:** Scattered components, dotted or broken lines between them, a visible gap (the silent bottleneck) shown as empty space or a question mark.
- **After:** Same components rearranged into a clear flow. Solid lines connecting them. The gap filled. New components (memory layer, correction loop) drawn in the accent color.
- Show both on the same image — left side "before" / right side "after" — or top/bottom.

## Component Visual Language

- **The human gate:** A component with a small person icon or a hand-drawn "stop" symbol. The system pauses here.
- **The silent bottleneck:** Shown as a gap, a dotted line, or a question mark in the before diagram.
- **The memory layer:** A component with a small brain or database icon sketched inside.
- **The correction loop:** An arrow that curves back to an earlier component — a visible feedback loop.
- **The bridge:** A component that connects a non-digital element (notebook, phone) to a digital element (CRM, WhatsApp).

## Mood

Authentic. "This is how I think." Not polished, not corporate. The kind of drawing that makes you trust the person because they're showing you their thinking process, not a finished deliverable.

## LinkedIn Dimensions

- Carousel slide: 1080 x 1350 px (4:5 ratio, portrait)
- Text must be readable at mobile size
- Title at top, diagram in center, "Todo es un sistema." at bottom
