# Visual Style Guide: Organic / Flow

## Identity

Systems as living things. Components aren't boxes — they're nodes in a network, like neurons, rivers, or root systems. This style reflects how Martin actually perceives systems: as interconnected flows, not rigid architectures. Warm, human, alive.

## Visual Characteristics

- **Line quality:** Flowing curves. No straight lines. Connections feel like rivers or neural pathways — they meander but always arrive.
- **Shapes:** Organic circles and rounded blobs for components. Not geometric — more like a mind map or biological diagram. Different sizes suggest importance.
- **Arrows:** Curved flowing lines. Varying thickness suggests flow volume (thicker = more traffic/data). No hard arrowheads — lines taper in the direction of flow.
- **Text:** Rounded, friendly sans-serif (Nunito, Quicksand feel). Labels float near components. Lowercase preferred — informal, approachable.
- **Colors:** Warm, natural palette. Earth tones (warm gray, terracotta, sage green, soft gold) on a light cream or warm white background. OR a gradient that suggests flow — components transition in color along the flow path.
- **Background:** Warm, slightly textured — like watercolor paper or recycled stock. Subtle, not distracting.
- **Layout:** Radial or organic placement. Main flow goes left-to-right or center-outward. Components cluster naturally, not on a grid. Feels like a map of a living system.

## How to Show Before/After

- **Before:** Isolated nodes. Thin, broken connections that don't quite reach. Muted, cold colors. The silent bottleneck is a gap — an island with no connection to the mainland. Feels fragmented.
- **After:** Same nodes now connected with thick, flowing lines. Warm colors bloom. New components (memory layer, correction loop) appear as new growth — like branches extending from a tree. The system feels alive and connected.
- Could use a metaphor: winter (before) → spring (after). Or dry riverbed (before) → flowing river (after).

## Component Visual Language

- **The human gate:** A node with a warm glow or a heart/person shape nested inside. This is where the human touch lives — the warmest point in the system.
- **The silent bottleneck:** A dried-up connection. A thin, cracked line that barely reaches the next node. Color: desaturated, cold gray. Visually obvious as the weakest point.
- **The memory layer:** A larger node with rings around it — like tree rings or ripples in water. Suggests accumulated knowledge.
- **The correction loop:** A flowing spiral that returns to an earlier node. Not a sharp arrow — a natural curve, like water finding its way back.
- **The bridge:** A flowing connection that transitions from warm (analog/human) to cool (digital/automated) colors along its length.

## Mood

Alive. "Systems aren't machines — they're ecosystems." This style says: the person who designed this sees connections others don't, and respects the human elements within the system. Appeals to founders who are intimidated by technical diagrams but can read a map.

## LinkedIn Dimensions

- Carousel slide: 1080 x 1350 px (4:5 ratio, portrait)
- Ensure readability — organic styles can get busy. Favor fewer components with clear labels over complexity.
- Title at top in warm tones, diagram in center, "Todo es un sistema." at bottom as a natural closing
